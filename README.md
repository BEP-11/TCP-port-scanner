
# 🔍 Rust Async TCP Scanner

[![Rust](https://img.shields.io/badge/Rust-2021-orange.svg)](https://www.rust-lang.org/)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](LICENSE)
[![Tokio Runtime](https://img.shields.io/badge/Runtime-Tokio%20v1-green.svg)](https://tokio.rs/)

Высокопроизводительный асинхронный сканер TCP-портов, написанный на **Rust**. Проект демонстрирует современные подходы к разработке инструментов кибербезопасности: конкурентное выполнение тысяч запросов, строгая безопасность памяти и элегантная обработка ошибок без единого `unsafe`.

---

## 📖 Описание
Инструмент выполняет быстрый проброс TCP-соединений в заданном диапазоне портов целевого хоста. Благодаря асинхронному рантайму `tokio` и управлению конкурентностью через семафоры, сканер способен обрабатывать тысячи портов одновременно, не истощая системные ресурсы и минимизируя вероятность срабатывания защитных механизмов (IDS/IPS).

---

## ✨ Возможности
- 🔹 **Безопасность памяти на уровне компилятора**: отсутствие переполнений буфера, висячих указателей и гонок данных
- 🔹 **Высокая пропускная способность**: асинхронные задачи `tokio::spawn` + ограничение одновременных соединений (`Arc<Semaphore>`)
- 🔹 **Удобный CLI-интерфейс**: парсинг аргументов через `clap`, автогенерация `--help`, валидация диапазонов
- 🔹 **Визуальный вывод**: цветовая индикация открытых портов прямо в терминале (`colored`)
- 🔹 **Лёгкая бинарная сборка**: статическая линковка, отсутствие времени выполнения зависимостей (zero-overhead)
